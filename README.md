# Overview

This project presents a hybrid AI-powered Clinical Decision Support System that combines rule-based symbolic reasoning with statistical machine learning to provide interpretable medical decision support.

Unlike conventional black-box prediction systems, the platform explains every recommendation through logical inference paths, confidence reasoning, and weighted decision factors.

The system demonstrates how Neuro-Symbolic AI can improve transparency, trust, and explainability in healthcare decision intelligence.
